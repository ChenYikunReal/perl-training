## Perl数据类型
Perl是一种弱类型语言，不需要定义类型，而其基本类型有：标量、数组、哈希。
- 标量：`$num = 123`
- 数组：`@arr = (1, 2, 3)`
- 哈希：`%hash = ('a'=>1,'b'=>2, 'c'=>3)`
